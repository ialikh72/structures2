Introduction

A Structure (struct) in C# is a value type used to store related data together. Structures are similar to classes, but they are lightweight and mainly used for small data objects.

Structures help organize data in a clean and efficient way.

Features of Structures in C#
Value type (stored in stack memory)
Can contain:
Variables
Methods
Constructors
Properties
Supports encapsulation
Faster for small data objects
Cannot inherit from another struct or class
Can implement interfaces
